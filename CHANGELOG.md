# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased] - 

## [0.1.2] - 2026-02-09

- Update README and add logo
- Make mass property editable
- Fixed hierarchy path matching 
- Add test case for state module
- Fix element sequence to avoid duplicated element names
- Fix size properties inconsistenies when editing elements

## [0.1.1] - 2026-01-28

- Switch to native add_cylinder for viser which is now supported since v1.0.17
- Fix project links
- Add test case for viser_utils

## [0.1.0] - 2026-01-27

 - Skip updating gripper pose 
 - Revise robot model loading: Add variant name and fallback robot model
 - Improve support for robot/gripper joint positions
 - Reduce unnecessary open3d imports and show info on first run

## [0.0.1] - 2026-01-24

 - Initial release
